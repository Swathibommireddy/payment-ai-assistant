# Payment AI Assistant
## Overview
Payment AI Assistant is an AI-powered customer support application designed to help customers resolve payment-related issues without waiting for a human support agent.
The application answers common payment questions, explains payment failures, assists with refunds, analyzes uploaded payment screenshots, and creates support tickets whenever AI cannot resolve the issue.
This project is built as a production-style Java backend application to demonstrate modern backend engineering skills.

## Objectives
- Reduce customer support workload
- Improve customer experience
- Provide instant AI-powered responses
- Automate payment support
- Demonstrate enterprise Java architecture
